## zh-CN

最简单的用法。

```HTML
  <nb-line-bar title="标题" height="46" [data]="data"></nb-line-bar>
```

## 描述
 柱状图折线图混合封装组件

## API


|    参数     | 参数 | 类型 |  默认值 |
| ---------- | --- |---- | --- |
| title |  标题 | string | null |
| data  |  指标值 | array<> | null |
| height  |  盒子高度 | number |auto  |
