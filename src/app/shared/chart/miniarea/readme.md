## zh-CN

最简单的用法。

```HTML
  <nb-miniarea title="标题" height="46" [data]="data"></nb-miniarea>
```

## 描述
 mini  面积图 暂时使用的折线图进行包装

## API


|    参数     | 参数 | 类型 |  默认值 |
| ---------- | --- |---- | --- |
| title |  标题 | string | null |
| data  |  指标值 | array<> | null |
| height  |  盒子高度 | number |auto  |
